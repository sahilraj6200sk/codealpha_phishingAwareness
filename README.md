# CodeAlpha_PhishingAwareness
# 🎣 Phishing Awareness Training

A cybersecurity awareness presentation created as **Task 2** for the **CodeAlpha Cyber Security Internship**. This project aims to educate users about phishing attacks, social engineering techniques, and best practices to stay safe online.

---

## 📌 Project Overview

Phishing is one of the most common cyber attacks used to steal sensitive information such as passwords, banking details, and personal data. This presentation explains how phishing works, how to identify phishing attempts, and how to protect yourself from becoming a victim.

---

## 🎯 Topics Covered

- Introduction to Phishing
- Types of Phishing Attacks
- How to Identify Phishing Emails
- Fake Websites and URL Analysis
- Social Engineering Techniques
- Real-World Phishing Examples
- Best Practices to Prevent Phishing
- What to Do If You Become a Victim
- Interactive Quiz
